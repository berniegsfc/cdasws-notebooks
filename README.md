# cdasws-notebooks
[cdasws](https://pypi.org/project/cdasws/) [Jupyter Notebooks](https://jupyter.org/)

Currently, this repository is mainly to support hosting the notebooks on [binder](https://binder.opensci.2i2c.cloud/).  But it may become more in the future.

Use <a href="https://binder.opensci.2i2c.cloud/v2/gh/berniegsfc/cdasws-notebooks/main" target="_blank">
<img src="https://binder.opensci.2i2c.cloud/badge_logo.svg"/></a> to launch the notebooks on binder.

To test CdasWsGenericExample.ipynb with AC_H2_MFI, use
<a href="https://binder.opensci.2i2c.cloud/v2/gh/berniegsfc/cdasws-notebooks/main?urlpath=notebooks%2FCdasWsGenericExample.ipynb%3Fdataset%3D%22AC_H2_MFI%22%26autorun%3Dtrue)" target="_blank">
<img src="https://binder.opensci.2i2c.cloud/badge_logo.svg"/></a>.

